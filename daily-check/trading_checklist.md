### 2.2 2% 목표에 맞는 진입 조건

| 조건 | 설명 |
|:----|:------|
| **1. 과매도** | 고가대비 -3% 이상 조정 = 단기 반등 기대 |
| **2. 시장 우호적** | 나스닥↑, 대장주 상승, 섹터 강세 → 반등 확률 상승 |
| **3. 고가까지 공간** | 최소 2% 이상의 고가 여유 필요 |
| **4. 외인/기관 참고** | 매도 중이어도 **과도한 물량이 아니면 큰 문제 아님** |
| **5. 손절 명확** | 진입가 대비 -1~2% 손절 = 안전망 |

## 3. 매매 일지 항목

### 3.1 일일 체크리스트

| 순서 | 항목 | 확인 |
|:---:|:-----|:----:|
| 1 | **나스닥 선물 방향** | 상승/하락/횡보 |
| 2 | **코스피 대장주**(삼전/하닉) | 상승/하락 |
| 3 | **섹터 강도**(반도체/바이오/2차전지) | 강함/보통/약함 |
| 4 | **장 분위기** | 갭업/갭다운/횡보 |
| 5 | **관심종목 고가대비** | -3% 미만 / -3~5% / -5% 이상 |
| 6 | **외인/기관 수급** | 매수/매도/중립 |
| 7 | **목표 수익률** | 2% (단타) or 5%+ (스윙) |
| 8 | **손절가 설정** | 진입가 대비 -1%, -2% |

### 3.2 진입 전 3초 체크

```
1. 고가대비 -3% 이상 빠졌나? (과매도=기회)
2. 나스닥/대장주/섹터가 우호적인가? (밀물인가?)
3. 고가까지 2% 이상 여유 있나? (2% 목표 가능?)
→ 3개 다 YES면 진입, 2개만 YES면 분할, 1개 이하면 패스
```

### 3.3 진입 후 관리

| 단계 | 행동 |
|:---:|:-----|
| **진입 직후** | 손절가 설정 (진입가 -1~2%) |
| **+1%** | 손절가를 진입가로 상향 (본전 방어) |
| **+2%** | 50% 청산, 나머지 손절가 +1%로 상향 |
| **+3%** | 30% 추가 청산, 나머지 손절가 +2% |
| **+5%** | 전량 청산 or 손절가 +3%로 올리고 추세 관찰 |
