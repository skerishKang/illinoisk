```
전날 15:40 📊 매매복기     ──┐

오늘 07:00 🇺🇸 미국장요약    ──┘       ↓
                              오늘 07:30 📋 매매준비

                              18:05 국장 마감 브리핑 → 21:00 watchlist 정리
```

## ⚠️ 중요: Kiwoom API 크론 = 두 스킬 필수

Kiwoom API 호출하는 크론은 반드시:
```python
skills: ["stock-market-analysis", "kiwoom-stock-api"]
```

`kiwoom-stock-api` 빠지면 180초 타임아웃 실패 (2026-05-29 실제 사례).
