## 📋 운영 규칙

### Core (11종목) — 상시 실시간 모니터링
- 30분봉 수집 크론 자동 실행 (`realtime_30min_YYYYMMDD.jsonl`)
- 장중 ka10001/ka10064/ka10002/ka90004 실시간 조회 우선순위 1순위
- 11:00/11:30/오후장 체크포인트에서 거래원/수급 우선 확인

### Extended (29종목) — 이벤트 드리븐 모니터링
- 분기보고서/공시/뉴스 발생 시 즉시 Core 승격 검토
- 섹터 로테이션 시 순환매 타겟으로 활용
- 장마감 후 일일 스크리닝(등락률/거래량/수급) 돌발 종목 체크

### 섹터 우선순위 (진입 시)
1. **전공정 장비** (Core 6 + Extended 12 = 18종목) — CapEx 직접 수혜, 외인 수급 가장 강함
2. **전공정 소재** (Core 1 + Extended 6 = 7종목) — 장비 후행, 순환매 시 고려
3. **후공정 테스트** (Core 3 + Extended 4 = 7종목) — 리노 1등주만, 나머지는 선별적
4. **대장주/파운드리** (4 + 3 = 7종목) — 시장 방향성 확인용, 주도주 동반 시만
