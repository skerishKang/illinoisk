# 📈 시장 패턴 (암기 필수)

## Pattern 1: 하이닉 솔로 랠리 ("닉스만 혼자 감")

**증상:** 하이닉 +1~3% / 삼전 flat~negative / 소부장 -3~7%
**진단:** False dawn. 하이닉의 HBM/AI 모멘텀이 장비주로 전이되지 않음.
**액션:** 삼전도 같이 갈 때까지 소부장 진입 금지.

## Pattern 2: 삼전 솔로 랠리 ("삼전만 혼자 감") ✅ 2026-05-29 추가

**증상:** 삼전 +4~5% AND/OR 하이닉 +2~4% / **ALL 소부장 RED**
**진단:** Pattern 1보다 심각. 대장주 둘 다 자금 빨아들이니 소부장 출혈 더 심함.
**액션:** 소부장 절대 진입 금지. 개별 호재 있는 종목만 예외적 고려.

## Pattern 3: 2차전지 + 현대차 Only Day

**증상:** 2차전지 + 현대차/기아만 강세 / 나머지 전종목 flat
**진단:** 올드머니 로테이션. 성장주에는 자금 안 들어옴.
**액션:** 현금 보류. 진입 금지.

## Pattern 4: 진짜 반도체 회복 ✅

**증상:** 삼전 + 하이닉 **둘 다** + / 소부장 follow
**진단:** 진짜 회복. 대장주가 장비주를 끌고 감.
**액션:** 소부장 진입 고려.

## Pattern 5: 두테-삼전 동조화 (2026-06-01 추가)

**증상:** 삼전 상승 시 두테(두산테스나)가 소부장 중 가장 먼저 반응. 두테의 등락 방향 = 삼전 방향과 일치하는 경향이 강함.
**진단:** 두테는 삼전 방향성을 확인하는 proxy 종목. 삼전이 오르면 두테가 먼저 반응하고, 나머지 소부장이 후행.
**액션:** 삼전 방향이 애매할 때 두테 움직임을 보면 판단 가능. 두테가 삼전 따라 안 가면 소부장 전체 모멘텀이 약한 신호.

## Pattern 6: "반도체만 빼고 다 오른다" — 전업종 동시 랠리 (2026-06-01 추가)

**증상:** 현대차(자동차)🟢 + LG전자(가전)🚀 + 두산(중공업/장비)🟢 + 네이버(IT플랫폼)🟢 **동시 상승**. 전혀 다른 업종의 대형주들이 일제히 오름. 반면 소부장(반도체 장비)은 4일 연속 하락 or 정체.
**발생:** 2026-06-01 실전 최초 관찰. 과거 전례 없음.
**진단:** 특정 테마 쏠림이 아닌 **자금의 광범위한 리밸런싱**으로 추정. 반도체 장비에서 이탈한 자금이 현대차·LG·두산·네이버 등 다양한 업종 대형주로 분산 유입. 반도체 장비를 제외한 전 업종이 수혜를 보는 구조.
**액션:** 소부장 반등을 기다리기보다, 이 패턴이 며칠 더 지속되는지 관찰 필요. 소부장 단타 진입은 리스크가 큼. 패턴 전환(소부장 반등) 시그널: 삼전 + 하이닉 동반 상승 + 코스닥 지수 반등.
**참고:** "소부장 4일 연속 하락"도 최근 전례 없음 (박사님 발언).

## Pattern 7: 이오-삼전·한미 상관관계 붕괴 (2026-06-01 추가)

**증상:** 삼전🟢 +3%↑ + 한미반도체🟢 +3%↑ = 1년간 이오테크닉스는 최소 +1.5~4% 상승이 정상이었으나, 2026-06-01 삼전🟢+3.31% + 한미🟢+3.19% 상황에서 이오는 🔴-3.06%로 **기존 상관관계 완전 붕괴**.
**추가 이상징후:** 이오 NXT 프리장에서 +1.20%(463,500)까지 반등 성공했으나, 정규장 09:00 시작 후 13분 만에 -18,000원(-3.06%) 폭락. 프리장 반등 → 정규장 매물 출회 패턴.
**발생:** 2026-06-01 실전 최초 관찰. 1년간 유지된 패턴 붕괴.
**진단:** 이오 개별 악재 가능성 or 소부장 내 자금 완전 이탈로 인한 체질 변화. 하이닉(-1.37%)이 약하긴 하지만 삼전+한미 조합이라면 기존 패턴으로는 최소 +1%는 받았어야 정상.
**액션:** 이오 상관관계 회복 여부 모니터링 필요. 며칠 더 지켜보고 패턴 복귀 or 확정적 붕괴 판단.

## Pattern 9: 유가 보합 리스크 오프 = 감정적 매도 → 반등 (2026-06-02 실전 검증)

**증상:** 글로벌 동시 하락 — 나선(나스닥 선물)↓ + 니케이↓ + 코스피↓ + 코스닥↓. 그런데 **유가(Brent)가 보합** (변동 없음).
**진단:** 유가가 안 움직이면 = 실제 원유 공급 차단 없음 = 실물 위기 아님. 주식 하락은 **감정적 패닉 매도**일 가능성 높음. 이란/지정학 리스크 헤드라인만으로 시장이 과민 반응.
**핵심 공식:** `유가 보합 + 지수 급락 = 감정적 매도 = 반등 가능`
**액션:** 장초반 패닉 매도 → 저점 확인 → 유가 여전히 보합이면 진입 검토. 반대매매 물량 소진 후 자연 반등 기대.
**반대 확인:** 유가도 같이 폭락하면 → 실물 위기 → 진입 금지, 현금 보류.

**2026-06-02 실전 사례:**
- 09:00 삼전 +5.73% 갭업 → 09:09 삼전 342,000 (-0.43%) 음전 → 09:30 361,000 (+3.5%) 반등
- 이오 저점 431,000 → 455,000 (+5.6%) 반등
- 주성 저점 175,000 → 196,400 (+12.2%) 반등
- 유가(Brent) 94.95 내내 보합 유지 — 유가가 안 움직였기 때문에 반등 가능했음
- **놓친 수익:** 종목당 4% 반등 × 500만원 = 최소 200만원

**교훈 (박사님):** "유가가 안움직였으니 회복이 가능할것이고" — 유가가 키. 유가 체크 없이 장초반 급락에 겁먹지 말 것.

**대응:** 겁먹지 말 것. 반대매매 물량 일시적 → 소진 후 자연 반등. 매수신호 확인 후 진입 고려. 단 하이닉 방향성도 함께 확인.
